# ⬡ AXIOM Studio v2.0

> AI-Powered Design, Presentation, 3D Creation & Social Media Automation Platform

---

## 🚀 Quick Start (One Click)

### Prerequisites
- **Python 3.10+** → [Download](https://www.python.org/downloads/)
- **Windows 10/11**

### Launch
```
Double-click → START_AXIOM.bat
```
That's it. The app installs everything, starts the server, and opens your browser.

---

## 🎯 What Can You Do?

| Feature | How to Access |
|:--------|:--------------|
| **Design Canvas** | Main workspace — drag nodes, edit text, shapes, images |
| **PPT Automation** | Click 📊 in sidebar → Enter topic → Download .pptx |
| **3D Model Engine** | Click 🎲 in sidebar → Generate & animate 3D models |
| **AI Chat** | Left panel → Type prompts for AI-generated content |
| **Social Assistant** | Click 🌌 in sidebar → Connect accounts, get post ideas |
| **Reddit Leads** | Nexus → Leads tab → Find potential customers |
| **Light/Dark Mode** | Click ☀️/🌙 at bottom of sidebar |
| **Timeline Editor** | Bottom panel → Multi-track video/audio/3D timeline |

---

## ⚡ Team Setup Notes

1. **API Key** is pre-configured in `.env` — no changes needed
2. **All features are FREE** during Beta — no payment required
3. **Each team member** should run `START_AXIOM.bat` on their own machine
4. **Blender** (optional) — Install from [blender.org](https://blender.org) for 3D model editing

---

## 🛠️ Manual Setup (if .bat doesn't work)

```bash
# Install Python packages
pip install -r requirements.txt

# Start the server
python main.py

# Open browser to http://localhost:8080
```
